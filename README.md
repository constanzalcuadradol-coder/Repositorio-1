# Repositorio-1
El primer ejericio de base de datos. Se intenta hacer online algo para lo que necesitaría docker desktop
1. Instalar Docker (docker.com)
2. Descargar AdventureWorks para Postgresql
(https://github.com/lorint/AdventureWorks-for-Postgres)
	(https://drive.google.com/file/d/1XjBp5fhJpUGmUAbhzDi3Utl9zoZ0PN8p/view?usp=sharing)
3. Descargar el tool DbClient
	(https://drive.google.com/file/d/1uc492I0uDZO2GeGSYthx5vP3tnJFJw5C/view?usp=sharing)
4. Levantar los servicios de docker 
5. Ejecutar el docker-compose de cada uno de los componentes 
(docker-compose up)
5. Acceder a http://localhost:8080
